Nonlinear Biplot Axes via B-Splines

Background

A biplot is a simultaneous display of the rows and columns of a data matrix in a low-dimensional space, typically used to visualise multivariate structure. Classical biplot axes are linear — they cannot faithfully represent nonlinear relationships between variables and the projected scores.

This work implements the spline-based nonlinear biplot axis method introduced by Groenen, Le Roux & Gardner-Lubbe (2015), which replaces straight calibrated axes with smooth curves parameterised by B-splines, enabling accurate reading of nonlinear data structure directly from the biplot display.

Original paper:
Groenen, P.J.F., Le Roux, N.J. & Gardner-Lubbe, S. (2015). Spline-based nonlinear biplots. Advances in Data Analysis and Classification, 9, 219–238.
DOI ↗

My Contribution

The original implementation was written in Fortran for computational performance. My work involved rewriting this core numerical code in C++, making it easier to maintain, extend, and integrate with modern R packages.

Fortran → C++ B-Splines Nonlinear Biplots Rcpp biplotEZ

Implementation in biplotEZ

The method is implemented in the open-source R package biplotEZ (Gower, Gardner-Lubbe & Le Roux, 2011). The core function lives in biplotEZ/R/plot2d.R as biplot.spline.axis().

R package:
Gower, J.C., Gardner-Lubbe, S. & Le Roux, N.J. (2011). biplotEZ — Biplots for Everyone.
CRAN ↗ GitHub ↗
